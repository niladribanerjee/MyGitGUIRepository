Objective¶

  • To identify the variables affecting house prices. e.g. area, number of rooms, bathrooms etc.
  • To create a linear model that quantatively relates to the house prices
  • To know the accuracy of the model

Steps¶

  • Reading, understanding and visualizing the data
  • Preparing the data for modeliing (train-test split, rescaling etc.)
  • Training the model
  • Residual analysis
  • Predictions and evaluation on the test set

# Rescale the variables
# why rescale area

# without rescaling, we might end up getting the following

# coef (area) = 0.0079
# coef (bedrooms) = 560

# can we say just by the coef, that bedrooms variable is of higher importance as predictor - No

# IT IS IMPORTANT TO HAVE THE VARIABLES AT A COMPARABLE SCALE SO THAT THE COEFFICIENTS ARE COMPARABLE

# MOREOVER Behind the scenes, there is gradient decent algorithm kicks in. The optimisation happens much faster if we rescale

# Two types of Scaling

#  a. Min-Max (normalization) : all data points will be converted in the range between 0 - 1
#  b. Standardization : (mean = 0, sigma = 1)

# Min-Max = (x - xmin) / (xmax - xmin)
# normalization = (x - mu) / sigma

# Min-Max scaling has an advantage of taking care of the outliers also. Hence this preferred than normalization

We could have the following situation

  • High p-value, High VIF = definately drop
  • High-Low cases:

  • Low p, Low VIF = definately retain
